## nvm 控制node版本切换

`nvm install v18.13.0` 安装18.13.0版本node

`nvm use 18.13.0` 使用18.13.0版本的node

`nvm list` 检查本地node版本`

`nvm --help`

`nvm alias v16 v16.16.0` 设置v16默认为v.16.16.0版本

`nvm use v16` 等于 `nvm use v16.16.0`
