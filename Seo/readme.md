## URL 导航

1. 导航包含关键词
2. URL不超过3级，不超过3个/即可

## TDK

1. title

80个字符以内，突出目标关键词，数量不超过5个

2. description
200个字符以内，关键词和标题一致，辅助关键词排名

3. keyword
100个字符以内，提高关键词密度，数量5个以内

## 内链优化

四处一词
标题-关键词-描述-超链接

## 代码优化

使用H1或者H2，只出现一词，突出页面重要标题，包含关键词

strong 强调关键词作用
